**XA**

```
GRAB 300
COPY F M
COPY F X
DROP
MAKE

@REP 10

@END

COPY X M

@END

LINK 800
COPY 10 T

MARK READ_LOOP

VOID F

@REP 10
COPY M F
@END

SUBI T 1 T

MARK WRITE_LOOP

VOID F

@END

@END

MARK WRITE
@REP 10
COPY M F
@END
```
